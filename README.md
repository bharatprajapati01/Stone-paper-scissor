# 🎮 Rock Paper Scissors Game (Python)

A simple command-line based **Rock, Paper, Scissors** game built using Python. Play against the computer and test your luck!

---

## 📌 Features

- User vs Computer gameplay  
- Random computer choices  
- Input validation  
- Continuous play until user quits  
- Beginner-friendly code  

---

## 🛠️ Requirements

- Python 3.10.8  
- No external libraries required (uses built-in `random` module)

---

## ▶️ How to Run

1. Save the file as `rock_paper_scissors.py`
2. Open terminal or command prompt
3. Navigate to the file location
4. Run the program:

```bash
python rock_paper_scissors.py
